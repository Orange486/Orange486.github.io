jackson转换出错一

Dec 04, 2019点击

今天在给后台向前端传数据时，报了一个：com.fasterxml.jackson.databind.exc.InvalidDefinitionException: No serializer found for class
的错。

在网上找到的解决办法是因为是实体类中有的字段的值为null，在实体类上加上注解 @JsonIgnoreProperties(value = { “hibernateLazyInitializer”, “handler” })就可以。但我的是因为用的lombok，那个实体类上忘记加上@Data注解。.嘻嘻。
